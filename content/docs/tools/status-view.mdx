## Status Types

|Status|Appearance|Use For|
|---|---|---|
|`created`|Gray indicator|Queued operations|
|`in_progress`|Pulsing orange + animated text|Active processing|
|`completed`|Green indicator|Successful operations|
|`error`|Red background + indicator|Failed operations|
