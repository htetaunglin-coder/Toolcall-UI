## Installation

<Callout title="Note:">
  This component uses [mijnui](https://mijnui.com) components under the hood. If you want to use it with shadcn/ui, you may need to make some changes.
</Callout>

<Steps>

  <Step>
  Install the required dependencies:
  
  ```package-install
  react-photo-view
  ```
  </Step>

  **Copy and paste the following code into your project.**

  <Step>
  Add the `image-view.tsx` file. 
  This file defines the `ImageView` component, which handles rendering image galleries with lightbox viewing, download functionality, and error states.

## Quick Start

```tsx
import ImageView from './components/image-view'

const images = [
  {
    id: "img-1",
    src: "https://example.com/image1.jpg",
    alt: "Description of image 1",
    width: 800,
    height: 600
  },
  {
    id: "img-2", 
    src: "https://example.com/image2.jpg",
    alt: "Description of image 2"
  }
]

<ImageView images={images} />
```
